
## Setup

Make sure to install dependencies:

```bash
pnpm install
```

## Environment Variables

Create a .env file in the root of the project and add the env variables. You can get them from the server

```
touch .env
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

## Production

Build the application for production:

```bash
pnpm build
```

Locally preview production build:

```bash
pnpm preview
```
